# 데이터 원본 (sources)

사이트 데이터의 **원본 목록** 보관용. 실제 데이터는 아래 코드 파일에 반영되어 있습니다.

| 원본 | 반영된 코드 | 비고 |
|------|------------|------|
| `keycaps-list.txt` | `src/data/keycaps.ts` | 키캡 세트 이름 목록 (원래 파일명 `스위치 종류.txt`였으나 실제 내용은 키캡) |
| `switches-naver-list.txt` | `src/data/switches.ts` | geonlab 네이버 스토어 판매 스위치 목록(원문). 중복/번들 정리 후 반영 |

> 코드의 데이터는 원본을 정리·보강한 것이라 원문과 1:1로 일치하지는 않습니다. 원문 추적이 필요할 때 참고용.
